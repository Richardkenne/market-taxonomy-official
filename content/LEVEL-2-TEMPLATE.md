## Parts

| Part | Element | Notes |
|------|---------|-------|
| 1 | Title | `[Market] — [Sub-Market]` |
| 2 | Sub-market definition | One sentence, max 2 lines |
| 3 | Sources explanation | One sentence |
| 4 | Sources | 2-column grid, 4–6 links |
| 5 | Segmentation explanation | Callout with green left border |
| 6 | Segments | Name · Description · Detail · Size (last two if available) |
| 7 | CTA | Optional — only if segments > 6 |
| 8 | How to choose a niche | Box, bridge toward Level 3 |
