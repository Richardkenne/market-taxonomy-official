# Level 3 Page — Template (Suggestion)

Level 3 is the deepest node — a specific, named segment.
At this point the user has navigated through 3 levels and is ready to act.
The page should be **information-dense but focused**, not exploratory.

```
[Home]  ← Back to [Sub-Market Name]

        ● Part A — Level 3

        [Segment Name]

        [Definition — one sentence, precise scope.]

────────────────────────────────────────

        [Sources explanation — one sentence.]

        SOURCES
        ┌──────────────────┬──────────────────┐
        │ → [Source 1]     │ → [Source 2]     │
        │ → [Source 3]     │ → [Source 4]     │
        └──────────────────┴──────────────────┘

────────────────────────────────────────

        MARKET CONTEXT
        ┌─────────────────────────────────────┐
        │ Size / Scale      [e.g. $2.5T+]     │
        │ Growth signal     [owner-occ. +X% · rental +Y%]  │
        │ Key players       [Name (role) · Name (role)]    │
        │ Geography         [e.g. Global/US]  │
        └─────────────────────────────────────┘

────────────────────────────────────────

        ┃ [Problem framing — one sentence on what
        ┃ makes this segment interesting for builders.]

        KNOWN PAIN POINTS
        · [Pain point 1]
        · [Pain point 2]
        · [Pain point 3]

────────────────────────────────────────

        ┌─────────────────────────────────────┐
        │ HOW TO CHOOSE A NICHE               │
        │ [Specific guidance for this segment.]│
        └─────────────────────────────────────┘

        [ Start Market Analysis → ]
          (CTA to Part B / wizard)
```

---

## Parts

| Part | Element | Notes |
|------|---------|-------|
| 1 | Title | Segment name only (no market prefix) |
| 2 | Definition | One sentence, precise scope |
| 3 | Sources explanation | One sentence |
| 4 | Sources | 2-column grid, 3–5 links |
| 5 | Market context | Size · Growth · Key players · Geography |
| 6 | Problem framing | Callout — why this segment for builders |
| 7 | Known pain points | 2–4 bullet points |
| 8 | How to choose a niche | Box, very specific to this segment |
| 9 | CTA | "Start Market Analysis →" → Part B / wizard |

---

## Notes

- No further sub-markets — this is a leaf node
- Market context fields are **optional individually** — leave blank if data not available
- Pain points are the **core value** of Level 3 — invest here
- CTA is **mandatory** — it's the only exit point toward Part B
