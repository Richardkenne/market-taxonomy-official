22 Software

Software markets are segmented by buyer type, deployment model, and functional category — each with distinct sales motions, pricing economics, and competitive dynamics.

We reference two types of sources: global classification standards used by investors and industry analysts, and market-intelligence research used to size segments and track software spending trends. Use these to validate market definitions before narrowing to a specific segment or geography.

 02

CRM & Revenue Operations

Platforms for managing customer relationships, sales pipelines, marketing automation, and revenue intelligence across the buyer lifecycle.

Includes: Sales CRM, Marketing Automation, Revenue Intelligence, Customer Success Platforms.

 03

Developer Tools & Platforms

IDEs, version control systems, CI/CD pipelines, and developer experience platforms that power software engineering workflows.

Includes: IDEs, Source Control, CI/CD, API Management, DevOps Platforms.

 04

Vertical SaaS

Industry-specific cloud software built for the unique workflows, compliance needs, and data models of a single vertical.

Includes: HealthTech SaaS, FinTech SaaS, PropTech SaaS, LegalTech, Construction Software.

 05

Collaboration & Productivity Software

Messaging, video conferencing, project management, and document collaboration tools that coordinate distributed teams.

Includes: Team Messaging, Video Conferencing, Project Management, Document Collaboration.

 06

AI & Automation Software

Machine learning platforms, robotic process automation, intelligent document processing, and generative AI tools augmenting business workflows.

Includes: ML Platforms, RPA, Intelligent Automation, Generative AI Tools.

 07

Infrastructure Software

Databases, middleware, operating systems, and system management software that form the foundation layer for applications.

Includes: Databases, Middleware, Operating Systems, Monitoring & Observability.

How to Choose a Niche

Focus on segments where workflow complexity creates switching costs and where recurring pain justifies subscription pricing. Examples: ERP migration tools for mid-market manufacturers moving from legacy systems; vertical SaaS for dental practices managing scheduling, billing, and insurance claims; AI-powered code review tools for platform engineering teams with large monorepos.
